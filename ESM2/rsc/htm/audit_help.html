Classification

Classification questions work exactly the same way as checklist questions, but with the addition that they will display a traffic light colour indicating the degree of difficulty for entering, working in and rescuing from that space. This is NOT to be confused with the degree of risk associated with the space; this should be picked up in the risk assessment which is uploaded separately.

Each of the selections you make will cause the program to allocate a classification colour.

These contribute to the overall classification colour for the audit, based on the following rules:

  • If the audit contains a single red, the overall status is red
  • If the audit contains no red, but a single amber the status is amber
  • If the audit contains no red and no amber the status is green
