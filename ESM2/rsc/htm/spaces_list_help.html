Space List

Once you have logged in you will see the main interface or 'Space List'

The Space List is your overview of all the spaces on your vessel or installation. Your task is to create an entry for each of the spaces within.
Each one you create will appear here in overview so you will be able to see the following:

ID A number attributed within the system for each entry
Name The name you give to the space e.g. 'Forepeak Tank'
Completion Status How far through the process of entering the information you are shown by a percentage completion bar.
Internal Classification An indication of the level of danger associated with the space
Entry Point Classification An indication of the level of danger associated with each entry point
S/O (Signed Off) Whether the completed entry has been signed off (verified)

Creating a Space
If you have sufficient user privileges you can create a Space. To create a Space click the 'Add' button.

This will generate the following box:

To begin creating an entry for your space you need to provide a name and basic description for the space and for the first entry point. All the information you provide at this point can be edited later, and further entry points can be added if required.  If there are multiple entry points it doesn’t really matter which you add at this point. Just start by adding the one with which you are most familiar.

Enter the information and click ‘Submit’

You should now see the entry in the Space List e.g.:

Now you need to start to build your space information by adding audits of the space and entry point as well as adding additional information such as photographs, comments and supporting documentation such as risks assessments. There are no strict rules for how you build up the information you need, but the process should work something like this:

  1. Create Space In System – Covered in the steps above
  2. Print Audit Sheets – Because the audits are conducted away from the computer you will need to use paper based forms
  3. Conduct Audits Internal/External  - These are done in the spaces themselves and their respective entry points. If possible and safe to do so you should also collect other relevant material such as photographs, sketches and experiential data.
  4. Enter Audit Data In System – Once you have performed the audits you can input the data you have collected.
  5. Augment Audit/Space –Enrich the space information by uploading any photographs drawings , documents and comments.
  6. Master Sign Off – Once all relevant information is complete, the Master (or person alocated similar responsibilities for the system) can sign off the space.

To begin competing each of the steps select the space with your mouse and click the button marked ‘details’

 
You can also shortcut this process by double clicking on the space in the list.
