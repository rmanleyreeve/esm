Audits

Printing Audit Forms

Before you can enter audit information you will need to print out the audit sheets to perform the audit.
You can print out a form for either the internal space by selecting “Space Audit’ or one for the entry point by selecting 'Entry Point Audit'. N.B. You will need one copy for each entry point. Clicking the buttons will open a PDF of the form, you can then select to print the copies you need via the PDF window.

Entering the Audit Information

Here you can select each audit to perform by clicking the 'Audit' button (see 'Audit Checklists' below for more information)

You can also here add/edit or delete entry points. Click on the 'Edit' button to make any changes or the 'Delete' button to remove the entry point.
Clicking the 'Add Entry Point' button will create the following window:

To create an Entry, enter the details as before and click ‘Submit’

Audit Checklists

Each audit is input into the system via the audit checklist. There are two types of Audit checklist:

  • Internal Space Checklist
  • Entry Point Checklist

Both are selected from the main space screen by selecting the 'Audit' button :

Both Checklists work the same way and are in two parts - Checklist and Classification Questions.

Photos

The system provides you with the ability to add photographs or any other image based media such as scanned drawings that will help to build information on the space and its entry points. To add one, simply click the 'Add' button, this will bring up the following window:


Select the file you wish to load into the system by clicking the 'Choose' button. This will bring up your standard Windows file explorer. You should browse for the image you want and select it as you would ordinarily.

Once selected you will see the name of the picture next to the 'Choose' button and can also add a name and description for your picture as in this example:

After clicking ‘Submit’ you will then see your photograph in the main Space Detail screen in thumbnail view.

It can then be viewed at any time by clicking on the photo thumbnail and clicking ‘View’

N.B. Double clicking on the photo will have the same result. It will open the following window:

Allowing you to edit the details by clicking the ‘Edit Details’ button or alternatively remove the photo from the system by clicking ‘Delete photo’
N.B. Moderated users will not be able to upload edit or delete photos and only have ‘read’ access.

Documents

The system provides you with the ability to upload relevant documents to the system. These could be information about the space or about your company specific working practices such as risk assessments, permit to work etc

To add a document simply click on the ‘add’ button. This will bring up your standard windows explorer. You should browse for the image you want and select it as you would ordinarily.

Once selected you will see then see the document in the main space screen in thumbnail view.

Select the relevant document and click the ‘view’ button to open it or remove it by clicking ‘delete’

Sign Off

A Space should not be seen as complete until it has been authorized or ‘signed off’. This means that a relevant person has evaluated all of the entered data and materials and is satisfied that all necessary information has been correctly entered and complies with the ships Safety Management System.
Only those with sign –off privileges (User with sign –off) can sign off a space. If you have such privileges you can sign off a space by clicking the ‘Authorize’ button. You should only authorize a space if you can see that the information that has been provided is correct, up to date and thoroughly in accordance with the ship's SMS.
